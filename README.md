# 🧠 MBTI Commons

Minimal decentralized MBTI personality sharing. No build step, no framework, just Gun.js.

## Usage

1. Open `https://YOURUSERNAME.github.io/mbti-commons/`
2. Enter name, select MBTI type
3. Click "Add to Commons" — syncs to all peers instantly
4. Download CSV anytime

## Stack

- Gun.js (P2P sync)
- Vanilla JS
- Single HTML file
